** Abbreviations

  - =PDecl= :: Program declaration. Essentially top of AST.
  - =PTerm= :: Program Term. The parameter for PDecl. i.e. =type PDecl = PDecl' PTerm=.
  - =FC= :: probably short for FileContext. Contains source positions produced by Parser.
